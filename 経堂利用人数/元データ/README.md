## エクスポート手順（Git に貼る）

1. 上記 URL のスプレッドシートを開く
2. **拡張機能 → Apps Script**
3. 左のファイル一覧で **`Code.gs`**（メイン）を開く → 全文コピー
4. このフォルダの **`Code.gs` に貼り付け（上書き）**
5. Apps Script に **`.gs` が他にもある場合**（例: `Dashboard.gs`）  
   → 同じファイル名で **この `元データ/` フォルダに新規作成** して貼る
6. **`../Code.gs` にも `Code.gs` と同じ内容をコピー**（作業用）
