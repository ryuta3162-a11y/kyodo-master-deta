## 拡張サービス

Apps Script **サービス** に **Google Calendar API**（Calendar）が追加されている必要があります。

**カレンダー同期対象**（`CAL_CONFIG.STAFF_LIST`）: 日下・蜂谷・中田・石田 美絵（経堂）

**イレギュラー勤務**（`STAFF_WEEKLY_OVERRIDE`）: 石田 美絵のみ — 火 11:00-17:00 / 水 13:00-17:00、それ以外は不在（火水でシートが休みなら休み優先）
