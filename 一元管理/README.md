# 一元管理（Git フォルダ名）

## Google ブック名（正式）

**経堂リニューアル　アイデアプール**

## これは何？

**共同データの司令塔** です。各ブックで集めた統計を、シートごとに集約していきます。

| シート例 | 集約する内容 |
|----------|--------------|
| **リフォーマー** | [経堂リフォーマーチェック表](../リニューアル/) から |
| **利用状況** | [経堂利用人数](../経堂利用人数/) FW/TM（IMPORTRANGE・リアルタイム） |
| **スタジオ** | [スタジオ/](../スタジオ/) ホットスタジオ（会員データ.gs・コード.gs・index.html） |
| **利用率** など | （今後） |

## GAS の貼り方（1ファイルだけ）

```
Apps Script → Code.gs を1つだけ
```

| 貼るファイル | 内容 |
|--------------|------|
| **[Code.gs](Code.gs)** | **全部入り**（リフォーマー＋利用状況） |

`_modules/` は Git 上の分割用です。**Apps Script には貼りません。**

## メニュー「共同データ」

| 項目 | 内容 |
|------|------|
| リフォーマーデータを更新 | 会員数コピー ＋ ⑦⑧⑨ |
| （確認）deta列を表示 | deta 列の確認 |
| 利用状況をリアルタイム連携 | 利用状況シートを IMPORTRANGE 設定 |
| 利用状況: FW ⇔ TM 切替 | フリーウェイト / トレッドミル切替 |
