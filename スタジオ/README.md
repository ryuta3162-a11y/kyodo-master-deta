# スタジオ（Git フォルダ名）

**ホットスタジオ利用者集計** 用の Apps Script 正本置き場です。

## Google ブック名（正式）

| 項目 | 値 |
|------|-----|
| ブック名 | **（未設定 — URL が分かれば追記）** |
| URL | （未設定） |
| ID | （未設定） |

## ファイル一覧

| ファイル | 役割 |
|----------|------|
| [Code.gs](Code.gs) | `aggregateAttendance()` — 月次シートから集計 → **「集計」** シートへ出力 |
| [WebApp.gs](WebApp.gs) | `doGet()` — ダッシュボード HTML を Web アプリとして公開 |
| [Index.html](Index.html) | React + Chart.js ダッシュボード（現状は `rawData` 手入力） |

## ブック内シート（Code.gs より）

| シート | 役割 |
|--------|------|
| **会員データ** | 会員番号・氏名・年齢 |
| **4月** / **5月** … | 月ごとの出席表（「○月」を含む名前） |
| **集計** | GAS が書き出すサマリー＋会員別参加回数 |

除外: シート名に `集計`・`原本`・`休講` を含むもの、`会員データ`

## フォルダ構成

```
スタジオ/
├── 元データ/          ← 凍結バックアップ（正本）
│   ├── Code.gs
│   ├── WebApp.gs
│   └── Index.html
├── Code.gs
├── WebApp.gs
└── Index.html
```

## ブックへの戻し方

対象ブック → **拡張機能 → Apps Script**

| Git | Apps Script |
|-----|-------------|
| `Code.gs` | `Code.gs` |
| `WebApp.gs` | 2つ目の `.gs`（名前は任意） |
| `Index.html` | **`index.html`**（`doGet` が `'index'` を参照） |

## 実行

| 操作 | 方法 |
|------|------|
| 集計更新 | スクリプトエディタで `aggregateAttendance` を実行 |
| ダッシュボード | Web アプリとしてデプロイ → `doGet` の URL |

※ Index.html の `rawData` は集計シートから手書き写し。自動連携は未実装。
