# スタジオ（Git フォルダ名）

**ホットスタジオ利用者集計** 用の Apps Script 正本置き場です。

## Google Apps Script のファイル名（正式）

**ブック側は次の3つだけ**です。Git のファイル名と **同じ名前** で貼り付けます。

| Apps Script 上の名前 | Git のファイル | 役割 |
|---------------------|----------------|------|
| **`会員データ.gs`** | [会員データ.gs](会員データ.gs) | **`aggregateAttendance()`** 集計本体（集計シート・G6の352人数など） |
| **`コード.gs`** | [コード.gs](コード.gs) | **`doGet()`** Webアプリ（ダッシュボードを開く入口） |
| **`index.html`** | [Index.html](Index.html) | ダッシュボード画面（React）※Apps Script 上は小文字 `index.html` |

```
Apps Script（3ファイル）
├── 会員データ.gs   ← ★集計・G6書込はここ（Code.gs ではない）
├── コード.gs       ← Webアプリ用（doGet）
└── index.html      ← 画面HTML（小文字 index）
```

※ 旧 Git 名 `Code.gs` / `WebApp.gs` / `2つ目.gs` は廃止しました。

## ブック内シート

| シート | 役割 |
|--------|------|
| **会員データ** | 会員番号・氏名・年齢（マスタ） |
| **4月** / **5月** … | 月ごとの出席表 |
| **集計** | GAS が書き出すサマリー＋名簿（A8〜） |

## 集計シートのセル（手入力・自動）

| セル / 列 | 内容 |
|-----------|------|
| **D6** | 閉店以降会員の氏名リスト（**手入力**・改行・読点区切り可） |
| **E6** | 閉店以降会員の総数（**手入力**） |
| **F列（6列目）全体** | **手入力専用** — GAS は読み書きしません |
| **G6** | 352番台の名簿ユニーク人数（**自動**） |
| **H6, I6, J6…** | 352番台会員の**月別参加人数**（4月→H, 5月→I … 出席表の352番号で判定・**自動**） |

名簿の月別参加回数は **H列から** 月ごとに出力されます（4月→H、5月→I、6月→J、7月→K …）。A〜D列は会員情報、E〜G列・F列は手入力用です。

## 実行

| 操作 | 方法 |
|------|------|
| 集計更新 | スプレッドシートを開き直すと **「ホットスタジオ」→「集計を更新」**（`onOpen` メニュー）。またはスクリプトエディタで `aggregateAttendance` を実行 |
| ダッシュボード | **コード.gs** を Webアプリとしてデプロイ → URL を開く |
