## 一元管理の考え方

- **データ** … 各 Google スプレッドシート
- **コード（GAS）** … このリポジトリのフォルダごとに1本（`Code.gs`）
- **1ブック = 1フォルダ** を基本に、リンクと GAS をセットで追記する

新しいブックを使うとき:

1. リポジトリにフォルダを作る（または既存フォルダを使う）
2. [スプレッドシート一覧.md](スプレッドシート一覧.md) に URL・ID・gid を登録
3. ブック側 Apps Script に `Code.gs` を貼り付け
