# kyodo-master-deta

JOYFIT24経堂の **複数スプレッドシート** と **GAS** を一元管理するリポジトリです。

## 一元管理

| 役割 | 場所 |
|------|------|
| データ・運用 | 各 Google スプレッドシート |
| GAS の正本 | このリポジトリの `各フォルダ/Code.gs` |
| 取りまとめ（目次） | **経堂リニューアル　アイデアプール** → Git [一元管理/](一元管理/) |
| ブックと GAS の対応表 | [マスタ/スプレッドシート一覧.md](マスタ/スプレッドシート一覧.md) |

**1ブック = 1フォルダ** … リンクを送ってもらえれば台帳とフォルダに登録し、GAS もここで共有します。

詳細リンク → [マスタ/各種リンク.md](マスタ/各種リンク.md)

## 構成

| パス | 内容 |
|------|------|
| [マスタ/](マスタ/) | 全ブック台帳（正式名称つき） |
| [一元管理/](一元管理/) | 経堂リニューアル　アイデアプール（目次） |
| [年間会員動向/](年間会員動向/) | 経堂データ　入会・退会・見学体験数など |
| [経堂利用人数/](経堂利用人数/) | 経堂利用人数集計　データ　FW　ランニングマシン |
| [経堂リニューアル/](経堂リニューアル/) | 年間会員動向【2026年度】 |
| [リニューアル/](リニューアル/) | 経堂リフォーマーチェック表 |
| [入会メール取込/](入会メール取込/) | 6カ月継続会員へ送る自動メールなど |
| [option/](option/) | 受付状況表(オプションGAS等) |
| `excel/` | ローカル用テンプレート（任意） |

## GAS の貼り付け先

アイデアプール → **拡張機能 → Apps Script → `Code.gs` 1ファイル**（[一元管理/Code.gs](一元管理/Code.gs) を全文貼り付け）
